# Job Employment Portal

A simple job employment web platform connecting job seekers with employers, built with HTML, CSS, JavaScript, and PHP.

## Features
- Home page with user registration (username, email, password)
- About page describing the platform's mission
- Job Details page showcasing available vacancies (stitching, decoration, mehndi, cleaning, driving, babysitting, photography, etc.) with a flip-card hover effect
- Contact page with phone and email
- PHP backend (`register.php`) for storing registrations in a MySQL database
